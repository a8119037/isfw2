# Contact Tracing

## Overview

PC教室におけるコンタクト・トレーシングのシステムです。

## Features

- 座席へのチェックイン・チェックアウトをQRコード経由で登録（スマホ可）
- 濃厚接触者の情報・接触時間・位置関係などを可視化
